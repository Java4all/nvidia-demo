# Training experiments (NeMo / local lab)

This folder is for **your** hands-on NeMo work — checkpoints, small datasets, notebook outputs — **not** required to run the triage agent.

- **Guide:** **[docs/NeMo_learning_lab.md](../docs/NeMo_learning_lab.md)** — Docker Jupyter, Primer notebook, optional pip env, links to Megatron-Bridge.
- **Artifacts:** put heavy outputs under **`experiments/`** (gitignored) so the repo stays small.

The main app stays inference-only in the repo root; NeMo training environments are separate by design.
